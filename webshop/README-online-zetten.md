# Website Online Zetten

Deze map is klaar voor GitHub Pages.

## Bestanden die mee moeten

- `index.html`
- `style.css`
- `script.js`
- de map `assets`

## Met GitHub Desktop

1. Open GitHub Desktop.
2. Kies `File` > `Add local repository`.
3. Selecteer deze map: `outputs/website`.
4. Als GitHub Desktop vraagt om een repository te maken, kies `Create a repository`.
5. Geef de repository een naam, bijvoorbeeld `kledingwinkel`.
6. Klik `Publish repository`.
7. Ga op GitHub naar `Settings` > `Pages`.
8. Kies bij `Branch` de branch `main`.
9. Klik `Save`.

Daarna krijg je meestal een link zoals:

`https://jouwnaam.github.io/kledingwinkel/`

## Echte betalingen

De winkelwagen is een demo. Voor echte betalingen heb je later een provider nodig zoals Shopify, Mollie of Stripe.
